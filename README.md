# 小小 - 订阅通知中心

> 作者：[小小](https://github.com/MuSan-Li)

## 项目简介

`订阅中心`，实现了用户注册、登录、发布订阅、查询订阅等基础功能。

## 技术选型

### 前端

主要运用阿里 Ant Design 生态：

- HTML + CSS + JavaScript 三件套
- React 开发框架
- Ant Design Pro 项目模板
- Ant Design 端组件库
- Umi 开发框架
- Umi Request 请求库

### 后端

- Java 编程语言
- Spring + SpringMVC + SpringBoot 框架
- MyBatis + MyBatis Plus 数据访问框架
- MySQL 数据库
- jUnit 单元测试库

### 部署

- 单机部署
- Nginx
